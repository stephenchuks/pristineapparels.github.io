About

My inspiration for this project stems from my experience running a unisex boutique called Pristine Apparels. Previously, I had created a WordPress website to handle online orders. Now, I'm taking this opportunity to develop a standalone web application version of the website, maintaining the same name and providing an enhanced user experience.

This project is a part of my portfolio for Holberton School. You can find more information about Holberton School and my other projects on the Holberton School GitHub repository.
